### TILTH ‚Üî Feed Loop Integration

**Feed Loop scenarios** become **TILTH scenarios:**
- Feed Loop provides the ethical dilemmas and characters
- TILTH provides the design exploration framework
- Same 8 scenarios, different perspectives:
  - Feed Loop: Role-play the negotiation

1. Read [START-HERE.md](START-HERE.md)

---

### Flow 3: Educator
**Goal:** Teach multi-level ethical design

**Session 1: Product Ethics (60 min)**
1. Show [HUB.html](HUB.html) ‚Äî navigation
2. Demo [tetrad-pad.html](tetrad-pad.html) Product mode
3. Students work in groups: pick scenario, design alternative
4. Share & discuss outputs

**Session 2: Organizational Ethics (60 min)**
1. Recap Product mode work
2. Demo [tetrad-pad.html](tetrad-pad.html) Designer mode
3. Students: same scenario, organizational solution
4. Compare artifact vs. system designs

**Session 3: Role-Play (60 min)**
1. Demo [cb-box.html](cb-box.html)
2. Students: pick roles, negotiate scenario
3. Debrief: How did negotiation inform design?

**Session 4: Synthesis (60 min)**
1. Review all three outputs (product, org, negotiation)
2. Discuss: What makes ethical products inevitable?
3. Students: Present final integrated analysis
