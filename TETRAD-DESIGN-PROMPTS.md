## ğŸ¯ Design Philosophy Behind These Prompts

Each prompt is structured to:

1. **Describe the current system** (what exists now)
2. **Surface the ethical tension** (the core conflict)
3. **Name the power dynamic** (who's pressuring whom)
4. **Set the design challenge** (what needs redesigning)

This gives the Tetrad AI enough context to generate options that:
- Address real stakeholder concerns
- Navigate power asymmetries
- Propose concrete alternatives
- Respect technical constraints
